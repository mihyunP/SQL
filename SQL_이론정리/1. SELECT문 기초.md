# SELECT문

```mysql
SELECT [ALL|DISTINCT] `컬럼명` AS 별명, 그룹 함수(`컬럼명`)-- 다중행 반환 OR 단일행 반환
FROM `테이블명` AS 별명
[WHERE `조건식`] -- 그룹화하기 전 조건
[GROUP BY `그룹화 할 컬럼명`]
[HAVING `조건식`] -- 그룹화 한 후 조건
[ORDER BY `컬럼명` ASC/DESC]; -- 정렬
-- ※MYSQL 실행 순서 : FROM -> WHERE -> GROUP BY -> HAVING -> ORDER BY -> SELECT
```
